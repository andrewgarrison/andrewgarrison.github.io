JQUERY

My favorite programming language that we learned was JQUERY. This is because it is a much more simplified version of Javascript and makes life A LOT easier. For example in Javascript to call a out an ID or class we had to use the statement - document.getElementByID("idname"); and in JQUERY it looks like this - $("#idname"); . As you can see from that small example that JQUERY is much more simplified, and can allow you to do much more in a less amount of time.

WEEK 2 - 10/4/14

In class this week we did more work in scratch and focused mainly on encryption and decryption. Encrypting and decrypting are ways to communicate with only certain people who know the key for the message you have written. The way that we used it in scratch was through the use of the alphabet. We simply asked the user to enter a message and we wrote a program that would encode the message with a specific shift in the alphabet (ex: a --> d and d-->g) then be able to decode the message and show our original message that was sent. For this project we worked in groups which was very enjoyable. I liked it because I was able to easily meet new people and I also got a little bit of an insight on how others think when programming to help myself when i'm working individually. On top of all we learned in scratch this week, it was equally as awesome to have the speaker come answer questions and talk to us about our ESF report. Learning about ourselves and our strengths from a business perspective is a different insight you don't usually get, I can tell i'm learning more about myself than I ever have before.
